# Zoom Clone

A real-time video conferencing application that allows users to create or join meeting rooms and communicate through live video and audio.

## 🚀 Features

- Create and join video meeting rooms
- Real-time video and audio communication
- Screen sharing during meetings
- In-meeting chat for participants
- Unique room IDs for secure meetings
- Multiple participants in a meeting
- Live peer-to-peer connection using WebRTC
- Meeting history to track previously joined rooms
- Responsive user interface

## 🛠 Tech Stack

**Frontend**

- React.js
- HTML5
- CSS3

**Backend**

- Node.js
- Express.js

**Real-Time Communication**

- WebRTC (peer-to-peer video/audio streaming)
- Socket.io (signaling & real-time events)

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/Mdirfan0786/Zoom-clone
```

Navigate to the project folder

```
cd zoom-clone
```

Install dependencies

```
npm install
```

Start the development server

```
npm start
```

## 🌐 Usage

1. Open the application in your browser
2. Create a new meeting room or join an existing one
3. Share the room ID with others
4. View previously joined meetings in the dashboard history
5. Start video conferencing in real time

## 📌 Future Improvements

- Meeting recording
- File sharing during meetings
- User authentication with OAuth
- Host controls (mute participants, remove users)
